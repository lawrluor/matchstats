About Smashstats

Smashstats tracks the stats of tournament-level players in the eSports Super Smash Bros. Melee scene. It parses data from tournament brackets and neatly presents statistics of tournament attendance, comprehensive set records, and head to head histories between players. The application intelligently recognizes and sorts players by tag and associates them with the sets and tournaments they have played in. You may search for a particular player or tournament using the search form in the navigation bar at the top of the page, or browse the database using the other links in the navigation bar.

Currently, Smashstats only accepts bracket data from Challonge, meaning that no match information can be automatically generated. Additionally, no player information is automatically entered besides their tag and set history; this information includes region, main, and secondary characters. The tournament database currently holds final bracket data for all Nationals since 2015.

If you are interested in this project and would like to contribute, contact the developers at: smashstats@gmail.com
